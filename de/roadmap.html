Roadmap und Versionen

Sprache en fr


Letztes Update der Seite: 21.06.2017

Hier unten finden Sie die Roadmap von GMF, aber auch die Links zu den Release Notes der verschiedenen Versionen.

Version 2.4

Diese Version ist die kommende GMF Version. Die unten definierten Funktionen sind Vorschläge, die von Teilnehmer der Usergruppe gemacht worden sind. Sie müssen noch durch den PSC und die Usergruppe validiert werden.

Funktion n° Beschreibung
2.4.1 Zeichnen in der Mobil App (readonly)
2.4.2 Ablösung der API durch IFRAMEs
2.4.3 Editieren: Reihenfolge
2.4.4 Editieren: readonly Attribute
2.4.5 Bearbeiten auf einem Tablette (touch)
2.4.6 CSV Export CSV aus dem Resultatfenster
2.4.7 Editieren von Mehrfach-Attributen
2.4.8 Deaktivierung der "is simple" Geometrieüberprüfung für einen Layer
2.4.9 Editieren durch Mapserver (WFS-T Edtion)
2.4.10 3D
2.4.11 Editieren - undo von kopieren und einfügen
2.4.12 Editieren - Menu um einen Punkt zu löschen

Version 2.3

Die Funktionen dieser Version sind jetzt festgesetzt und die Entwicklungen sollten bald anfangen.

Diese Version sollte bis November-Dezember verfügbar sein.

Funktion n° Beschreibung
2.3.1 Integration des WMS / KML Browser + Permalink
2.3.2 Rotation der Karte
2.3.3 Popup-Fenster auf der Karte verschieben und skalieren
2.3.4 Administrationsschnittstelle
2.3.5 Optimierung des Resultate-Fenster
2.3.6 Flush / no flush vom Layertree auf der Mobile-App
2.3.7 Transparenz der Layers in der Mobil-App

2.2.6 Userscale im Druck-Modul
2.2.7 Kompatibilität mit dem Android Webrowser
2.2.8 Definierte Standard-Transparenz von WMS / WMTS Ebenen
