Roadmap & version

Langue de en


Dernière mise à jour de la page: 21.06.2017

Ci-dessous vous trouverez la roadmap de GMF ainsi que les notes listant les nouveautés fonctionnelles de chaque nouvelle version.

Version 2.4

C'est la future version de GMF. Les fonctions définies ci-dessous sont des propositions faites par les membres de la communauté. Elles doivent encore être validées par la communauté et le PSC.

Fonction n° Description
2.4.1 Dessins dans le mobile (readonly)
2.4.2 Remplacement de l'API par des IFRAME
2.4.3 Edition: ordre des colonnes
2.4.4 Edition: attributs en lecture seule
2.4.5 Edition sur tablette (touch)
2.4.6 Export CSV depuis la fenêtre de résultat
2.4.7 Edition de masse d'attributs
2.4.8 Désactivation de la vérification de la géométrie "is simple" pour une couche
2.4.9 Restriction de l'édition de par les requêtes Mapserver (édition WFS-T)
2.4.10 3D
2.4.11 Edition - undo de copier-coller
2.4.12 Edition - menu pour supprimer un vertex

Version 2.3

Les fonctions de cette version sont maintenant fixes (plus de modifications) et le développement va prochainement débuter.

Cette version devrait être disponible d'ici novembre-décembre 2017.

Fonction n° Description
2.3.1 Intégration du navigateur WMS / KML + permalien
2.3.2 Rotation de la carte
2.3.3 Déplacement de fenêtre et redimensionnement
2.3.4 Interface d'administration
2.3.5 Optimisation de la fenêtre de résultats
2.3.6 Flush / no flush de l'arbre des couches dans la version mobile
2.3.7 Opacité pour les couches dans la version mobile

Version 2.2

Toutes les fonctions de la version 2.2 ont été développées.

Fonction n° Description
2.2.1 Requêteur et filtres
2.2.2 Requête WFS - prise en compte le temps
2.2.3 Texte pour une recherche vide
2.2.4 Intégration Streetview
2.2.5 Bouton de ré-initialisation de l'arbre des couches
2.2.6 Permettre à l'utilisateur de définir sa propre échelle d'impression
2.2.7 Compatibilité pour le navigateur Android
2.2.8 Transparences pré-définies pour le WMS / WMTS
